# 🎓 LMS - Learning Management System (MERN Stack)

A full-stack Learning Management System built using the MERN stack.  
This platform allows users to register, login, access courses, and watch YouTube-based learning videos.

## 🚀 Live Demo

 (https://lms-mern-green.vercel.app/#/login)

---

## 📌 Features

- 🔐 User Authentication (JWT based login/register)
- 👨‍🎓 Student & Admin Roles
- 📚 Course Creation & Management
- 🎥 YouTube Video Integration
- ⏯ Video Resume Support
- 📊 Progress Tracking (if added)
- 🌐 Fully Deployed (Vercel + Render)

---

## 🛠 Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Axios
- React Router DOM

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt
